# Uber India Dashboard

## Overview
This project analyzes Uber ride data across India using Power BI. The dashboard provides insights into booking patterns, revenue performance, trip trends, customer behavior, and operational metrics through interactive visualizations.

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Key Features
- Booking Trend Analysis
- Revenue Performance Tracking
- Ride Category Analysis
- Customer Insights
- Interactive KPI Dashboard
- Dynamic Filters and Drilldowns

## Skills Demonstrated
- Data Cleaning
- Data Transformation
- Data Visualization
- Dashboard Development
- Business Intelligence Reporting
- KPI Analysis
